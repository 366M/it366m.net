PL - Co oznacza błąd 404? Jeżeli na stronie pojawia się komunikat 404 page not found lub podobny, oznacza to, że wyszukiwarka może połączyć się
z serwerem (w przeciwnym przypadku wyświetliłby się komunikat
server not found), lecz ten nie znajduje właściwego pliku odpowiadającego za działanie danej podstrony.

ANG - What does the 404 error mean? If a 404 page not found or similar message appears on the page, it means that the search engine can
connect to the server (otherwise the server not found message would
be displayed), but the server cannot find the correct file responsible for the operation of the subpage.
