Error 501
PL - Błąd 501 (Not Implemented) to jedna z kilkudziesięciu odpowiedzi HTTP, która oznacza, że serwer nie obsługuje funkcji wymaganych do
spełnienia danego żądania. Pojawia się wtedy, gdy serwer nie rozpoznaje metody żądania.
Jak powszechnie wiadomo, kody 5xx to błędy po stronie serwera, a kody 4xx – po stronie klienta.
Wróć do strony głównej: it366m.net
ANG - Error 501 (Not Implemented) is one of dozens of HTTP responses, which means that the server does not support the functions required
to fulfill the request. Occurs when the server does not recognize the
request method. As is well known, 5xx codes are server-side errors and 4xx codes are client-side errors.
